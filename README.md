
# 地址信息提取工具

一个基于Flask的地址信息提取工具，可以从包含姓名、电话和地址的文本中提取结构化信息。

## 功能特点

- 📱 提取手机号码
- 👤 提取收件人姓名
- 🏙️ 提取省市区信息
- 🏠 提取街道和详细地址
- 🌍 支持中英文地址
- 🎯 智能识别地址各组成部分

## 快速开始

### 环境要求

- Python 3.6+
- Flask

### 安装依赖

```bash
pip install flask
```

### 启动服务

```bash
python app.py
```

服务将运行在：http://localhost:3067

## 使用方法

### 网页界面

1. 打开浏览器访问 http://localhost:3067
2. 在文本框中输入包含地址信息的文本
3. 点击"提取信息"按钮
4. 查看提取结果

### API 接口

```
POST /extract
Content-Type: application/json

{
    "address": "张三 13800138000 广东省深圳市南山区科技园路123号"
}
```

#### 响应示例

```json
{
    "name": "张三",
    "phone": "13800138000",
    "province": "广东省",
    "city": "深圳市",
    "district": "南山区",
    "detail": "科技园路123号"
}
```

## 示例

### 输入
```
李四 13900139000 北京市朝阳区建国路88号现代城A座
```

### 输出
```json
{
    "name": "李四",
    "phone": "13900139000",
    "province": "北京市",
    "city": "朝阳区",
    "detail": "建国路88号现代城A座"
}
```

## 技术说明

- 使用正则表达式进行文本匹配和提取
- 采用前后端分离架构
- 响应式设计，适配移动端和桌面端
- 提供RESTful API接口
